# Film & Series Backend API

.NET for managing Films and Series.  
This project focuses on backend development and includes database integration (SQL).

## Features
- CRUD operations for Films and Series
- Filter, search, and sort functionality
- Database seeding using SQL script
- Front is not include 

## Technologies
- ASP.NET Core
- Entity Framework Core
- SQL Server / PostgreSQL
- RESTful API

## Project Structure
- `Controllers/` → API endpoints
- `Entities/` → Domain models
- `Data/` → Database context
- `Program.cs` → App startup
- `appsettings.json` → Configuration

## Database
- `film_db.sql` → Sample database script to populate Films & Series
